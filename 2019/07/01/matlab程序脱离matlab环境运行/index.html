matlab程序脱离matlab环境运行
发布于：2019-07-01 18:03:08 标签：/ matlab / 访问：

我的需求来源于：

  • 想用Graph500的图生成器，生成两个大规模的图数据集。在官方github源码中发现了.m文件，测试发现可在matlab中运行生成图数据集。

  • 但是，由于我的PC内存容量有限（8GB），最多生成（28，1）的生成图（4GB），即节点数为2^28，边数比节点数为1的子图。

  • 我的需求是分别生成一个（30，32）— 128GB 和（31，128）— 1TB的生成图，至少需要能够生成（31，1）的子图，然后拼起来，即32GB，内存需求最少为64GB。

  • 所以，可以将kronecker_generator.m转成exe可执行文件直接放到64GB服务器上去执行，免去在服务器上装matlab。

在有matlab的电脑生成.exe可执行文件

  1. 生成kronecker_generator.exe文件

  2. 测试：打开cmd命令行，进入kronecker_generator.exe所在目录，运行

  3. 带参数运行
    （1）首先在kronecker_generator.exe中添加代码

function ij = kronecker_generator (SCALE, edgefactor)
if (ischar(SCALE))
    SCALE = str2num(SCALE);
end
if (ischar(edgefactor))
    edgefactor = str2num(edgefactor);
end

（2）cmd中测试运行

在另一台电脑上运行

传输可执行文件到目标电脑后执行

此方法只能用于另一个Windows系统下脱离MATLAB环境运行，Linux中不能直接运行.exe。

要考虑在Linux环境下脱离MATLAB环境运行，需要尝试其他方法，比如将m文件转化为c/c++语言文件，但是还没尝试过。
